## Workflow-Übersicht

```
Planung          → Vorbereitung       → Durchführung    → Abschluss
──────────────      ───────────────      ─────────────     ─────────
Ticket eröffnen    Runbook lesen        Schritt folgen    Docs abgeben
Kunde im Katalog   Checkliste wählen    Befehle kopieren  Ticket schließen
Fenster abstimmen  Metadaten ausfüllen  Punkte abhaken    Snapshots anhängen
                   Pre-Go prüfen        Snapshots machen
```
